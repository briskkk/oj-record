### 9. 提取不重复的整数

一看到重复就要联想到使用`set`

* 题目：输入一个int型整数，按照从右向左的阅读顺序，返回一个不含重复数字的新的整数
* 输入：输入一个int型整数
* 输出：按照从右向左的阅读顺序，返回一个不含重复数字的新的整数
* 示例

>输入
>
>> 9876673
>
>输出
>
>> 37689

* 解答：

1. 

```python
    a = input()[::-1]
    b = ''
    for i in a:
        if i not in b:
            b += i
    print(b)
```

2. 字典法：

```python
from collections import OrderedDict
while True:
    try:
        in_list = [i for i in input().strip()][::-1]
        dic = OrderedDict()
        for i in in_list:
            if i not in dic.keys():
                dic[i] = 1
            else:
                dic[i] += 1
        out = ''
        for i in dic.keys():
            out += i
        print(out)
    except:
        break
```

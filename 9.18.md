# 2. 

给定一个字符串，将该字符串中的大写字母全部转为小写，并将空格转为数字0，最后将处理后的新字符串倒序输出。

* 输入描述

第一行输入：原始字符串，包括大小写字母，数字和空格，不含其他字符，用例保证，不用检查，字符串中最多1000个字符。

* 输出描述

处理后的字符串

* 示例

```pascal
input: abcdefg
output: gfedcba
```

* 解题

```python
s = input().strip()
s = s.lower()
s = s.replace(" ","0")
print(s[::-1])
```
